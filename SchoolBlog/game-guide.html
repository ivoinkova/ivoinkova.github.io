Навчайся із задоволенням!

Git Game

Git Game - консольна гра, пройшовши яку ви повністю опануєте цією системою контролю версій. Суть гри полягає в використанні різних команд, щоб знайти підказки для вирішення різних загадок.

CSS Diner

Відмінна гра для знайомства з CSS-селекторами. Гравцеві дається стіл з посудом, заданий анимированной HTML-формою, і конкретний предмет або предмети, які потрібно взяти зі столу. Вивчено буде все, від основ до ~ і: first-child.

FLEXBOX FROGGY

Логічна гра, в якій ви повинні допомогти кольоровим жабам дістатися до їх улюблених латаття. Робити це вам потрібно буде за допомогою CSS Flexbox, що дуже допоможе розвинути навички верстки. Підказки покажуть, як краще вирішити кожну задачу, але фінальне рішення залишається за вами - способів вирішення завжди кілька.

Відповіді :https://www.agreatdaytocode.com/flexbox-froggy/

CSS Grid Garden - навчальна гра по сіткам

Збираєтеся вчити CSS сітки (grids), але все ще лінь читати книжки? Тоді спробуйте гру CSS Grid Garden, за 28 рівнів навчитеся!.

Code Monkey

У цій грі ви навчитеся не тільки кодувати, але і ловити банани! На заході вчителі часто використовують Code Monkey для уроків програмування у дітей. Навчають програмувати на CoffeScript і Python. Вчителям не потрібен досвід програмування, щоб навчити цьому навику дітей з використанням CodeMonkey. Гра містить навчальний план, який включає в себе 35 детальних планів онлайн і оффлайн уроків. Вчителі також мають ексклюзивний доступ до панелі управління, де вони можуть відслідковувати прогрес і досягнення учня, а також розміщувати роботи учнів для обговорення всім класом.

Code Hunt

У грі Code Hunt ви, мисливець за кодом, виправляєте код, щоб він повертав потрібний результат. Для проходження доступні 14 рівнів, в кожному з яких є по кілька завдань. Гра підійде тим, хто хоче вивчити Java або C #.

Code Combat

CodeCombat - це платформа для студентів і школярів, що дозволяє вивчити комп'ютерні науки, граючи в справжні гри. Барвиста анімація і цікавий сюжет, клани, взаємодія з тисячами інших гравців по всьому світу - ось що вам належить побачити. Підтримуються такі мови, як Java, JavaScript, Python, Lua, CoffeeScript. Є російська локалізація.

Hex Invaders

У цій версії класичної аркади на світ нападають три інопланетянина різних кольорів. Для того щоб врятувати людство, гравець повинен швидко переводити шістнадцяткове значення кольору на людську мову і збивати прибульців.
